# 🌊 Батискаф: Глубоководная экспедиция

Браузерная аркада на чистом JavaScript, в которой вы управляете батискафом, исследуете океанические глубины, собираете ценные ресурсы и уклоняетесь от смертельных угроз.

## 🎮 Особенности
- **Динамичный геймплей**: Плавное управление, система столкновений и прогрессия под водой.
- **Три уровня сложности**: Влияют на скорость врагов, частоту появления предметов и расход кислорода.
- **Сонар**: Тактический инструмент, замедляющий мины и подсвечивающий их на короткое время.
- **Опасности**: Морские мины разного калибра и горизонтально перемещающиеся акулы.
- **Система рекордов**: Автоматическое сохранение результатов с фильтрацией по сложности и экспортом в JSON.
- **Визуальные эффекты**: Частицы при столкновениях, тряска камеры, фоновые пузырьки и адаптивный интерфейс.

## 🚀 Как запустить
1. Скачайте или клонируйте репозиторий.
2. Убедитесь, что в корневой папке находятся изображения: `bg.jpg`, `player.png`, `shark.png`, `bomb.png`, `crystall.png`, `healer.png`, `kislorod.png`, `sonar.png`.
3. Откройте `index.html` в браузере (рекомендуется Google Chrome или Edge).
4. Введите имя, выберите сложность и начните погружение.

## 🎛️ Управление
| Клавиша | Действие |
|---------|----------|
| `W` / `↑` | Движение вверх |
| `S` / `↓` | Движение вниз |
| `A` / `←` | Движение влево |
| `D` / `→` | Движение вправо |
| `E` | Активировать сонар (при наличии заряда) |
| `P` | Пауза / Продолжить |

## 🛠️ Технологии
- HTML5 Canvas API
- CSS3 (Flexbox, Animations, Variables, Media Queries)
- JavaScript ES6+ (Classes, `requestAnimationFrame`, `localStorage`, Modules-free architecture)
- Zero Dependencies (без фреймворков и библиотек)
